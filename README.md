# DJ FRAPCORE

## Installation

* `git clone https://github.com/MarconatoB/DJ-FRAPCORE.git`

* `pip install ffmpeg`

* `Télécharger ffmpeg et ajouter le dossier bin de ffmpeg à ton PATH ( jsp comment faire sur mac / raspberry pi)`

* `python dj.py`
